4.2 R graphs galleries

I have seen you people discussed a lot about how to use graphs. This link, https://r-graph-gallery.com/ will give you quick tutorial how to create those plots. Please click on the plot that you want, for example, Barplot, then it will take you to the page with many bar plots and instruction. This is the link that I got after I chose the most basic barplot from the main page: [https://r-graph-gallery.com/218-basic-barplots-with-ggplot2.html]

Another library that we can use to create plot is Plotly, here is the link to the Plotly gallery: https://plotly.com/r/. Similarly, click the plot that you want to create and you have the code on your hands. I personally use ggplot2 more frequenly then PLotly.
